9. Strava 연동

이 섹션의 목적

Strava에 쌓인 기존 라이딩 데이터를 오라이더로 가져와서 통합 분석하는 것. Strava에서 기록한 활동도 세그먼트 매칭, 피트니스 분석 등 오라이더 웹의 모든 기능을 활용할 수 있게 됩니다.

9.1 Strava 연결하기

해야 할 것

설정 → 연결된 앱
→
Strava 카드의 연결 클릭
→
Strava 인증 페이지에서 승인
→
연결 완료

확인할 것

연결이 완료되면 연결된 앱 탭의 Strava 카드에 초록색 점과 함께 연결됨 표시가 나타나고, 상단에 "N개 연결됨" 개수가 갱신됩니다. 계정 & 프로필 탭 상단 프로필 카드에도 Strava 연결됨 배지가 표시됩니다.

결과

Strava 계정이 연결되어, 과거 활동 가져오기와 자동 업로드가 가능한 상태가 됩니다.

앱과 웹의 Strava 연동은 자동 동기화됩니다. 웹에서 연결하면 앱에서도 연결 상태가 반영됩니다.

9.2 과거 활동 가져오기

해야 할 것

연결 후 연결된 앱 탭에 나타나는 Strava 동기화 옵션 카드에서, 과거 활동 가져오기 행의 가져오기 버튼을 누릅니다. 이동한 페이지에서 가져오기를 시작하면 Strava에 쌓인 기존 활동을 한꺼번에 불러옵니다.

확인할 것

  • 단계 표시(시작 → 진행 → 리포트)와 진행 상황
  • 가져온 활동 수 / Strava 전체 활동 수
  • 가져온 활동이 대시보드 피드에 추가됨

해석 기준

가져온 활동은 일반 활동과 동일하게 취급됩니다:

  • 세그먼트 자동 매칭 대상
  • 피트니스 분석에 반영 (파워/심박 데이터 포함 시)
  • 주간 통계에 합산

결과

Strava에만 있던 과거 기록이 오라이더에 통합되어, 전체 라이딩 역사를 하나의 플랫폼에서 관리합니다. 가져오기를 한 번 완료하면 같은 버튼이 리포트 보기로 바뀌어 결과를 다시 확인할 수 있습니다.

9.3 자동 업로드

해야 할 것

연결된 앱 → Strava 동기화 옵션 카드에서 자동 업로드 스위치를 켭니다.

결과

활동 종료 시 새 활동이 Strava에 자동으로 업로드됩니다.

9.4 연결 해제

해야 할 것

설정 → 연결된 앱 → Strava 카드의 연결 해제를 클릭하고, 확인 대화상자에서 승인합니다.

결과

Strava 연동이 해제됩니다. 이미 가져온 활동 데이터는 그대로 유지됩니다.

연결 해제 후에는 새로운 Strava 활동이 자동으로 동기화되지 않습니다.
