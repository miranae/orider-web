5. 지도

오라이더는 다양한 지도 스타일을 제공하며, 오프라인 지도를 미리 다운로드해서 데이터 없이도 사용할 수 있습니다.

5.1 지도 유형

라이딩 환경에 맞는 지도 스타일을 선택하세요. 설정 → 지도에서 변경할 수 있습니다.

유형설명추천 상황
Standard (Mapnik)기본 OpenStreetMap 스타일일반 주행
Cycle (CyclOSM)자전거 도로 강조, 경사 표시자전거 전용 도로 확인
Terrain (OpenTopoMap)등고선, 지형 표시산악/업힐 코스
MapBox고품질 벡터 지도깔끔한 시각 선호

5.2 오프라인 지도

셀룰러 데이터 없이 라이딩하거나 산간 지역에서 신호가 약할 때를 대비해 지도를 미리 다운로드하세요.

  1. 설정 → 지도 → 오프라인 지도로 이동합니다
  2. 원하는 지역을 선택합니다 (서울 / 경기 / 전국)
  3. 다운로드가 시작되면 진행률, 속도, 예상 소요 시간이 표시됩니다
  4. 완료된 지역은 체크 표시, 일부 다운로드 시 진행 표시가 나타납니다
전국 지도는 저장 공간을 상당히 사용합니다. Wi-Fi 환경에서 다운로드를 권장합니다.

5.3 헤딩 잠금

설정 → 지도에서 헤딩 잠금을 활성화하면 지도가 항상 이동 방향을 위로 향하도록 회전합니다. 내비게이션처럼 사용하기 편리합니다. 비활성화 시 북쪽이 항상 위를 향합니다.

경로를 따라가며 주행할 때는 헤딩 잠금을 켜는 것이 직관적입니다. 전체 경로를 조망하고 싶을 때는 꺼두세요.
