9. Strava 연동

Strava를 사용하고 있다면 오라이더와 연동하여 주행 기록을 자동으로 업로드하거나, 기존 Strava 활동을 가져올 수 있습니다.

연결하기

  1. 설정 → 계정으로 이동합니다
  2. STRAVA 섹션의 연결 버튼을 탭합니다
  3. Strava 로그인 페이지로 이동합니다 (OAuth 프록시 경유)
  4. 인증이 완료되면 연결된 계정 정보가 표시됩니다

자동 업로드

자동 업로드를 활성화하면 주행 저장 시 자동으로 Strava에 업로드됩니다. 별도 조작 없이 라이딩을 끝내면 바로 Strava에 기록이 올라갑니다.

Strava에서 가져오기

Strava에서 가져오기 버튼을 탭하면 Strava의 기존 활동을 앱으로 가져옵니다. 진행률 표시와 함께 새 활동 수가 표시됩니다.

Strava 활동 상세

기록 → 세션 상세에서 Strava에 업로드된 활동의 상세 정보를 확인할 수 있습니다.

연결 해제는 STRAVA 섹션의 "연결 해제" 버튼을 탭합니다.
