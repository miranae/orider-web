7. ClimbPro (오르막 감지)

긴 오르막에서 "정상까지 얼마나 남았지?"는 가장 절실한 궁금증입니다. ClimbPro는 경로 데이터를 분석하여 각 오르막의 진행 상황을 실시간으로 알려줍니다.

사용 방법

  1. GPX 경로를 불러옵니다 (6. 경로 참고)
  2. 경로에 오르막이 감지되면 주행 화면에 ClimbPro 페이지가 자동으로 추가됩니다
  3. 오르막에 진입하면 ClimbPro 페이지로 자동 전환되며, 자세한 정보가 표시됩니다

오르막 카테고리

오르막은 거리와 경사에 따라 자동으로 분류됩니다. (점수 = 거리(m) x 평균경사(%))

카테고리점수예시
HC (Hors Categorie)80,000 이상극한 난이도 고개
Cat 164,000 ~ 80,000장거리 급경사
Cat 232,000 ~ 64,000중급 오르막
Cat 316,000 ~ 32,000중간 오르막
Cat 48,000 ~ 16,000짧은 오르막

ClimbPro 화면에 표시되는 정보

  • 현재 오르막 이름 및 카테고리 (색상 구분)
  • 진행률 (%) 및 남은 거리/고도
  • 실시간 경사도 프로필 그래프
  • 경사 구간별 색상 표시
  • 다음 오르막까지의 거리
데이터 화면에도 ClimbPro 필드를 배치할 수 있어서, 지도 화면을 보면서도 오르막 정보를 동시에 확인할 수 있습니다.

7.2 PreClimb (자동 오르막 감지)

경로를 불러오지 않아도 괜찮습니다. PreClimb은 현재 위치와 이동 방향을 기반으로 서버에서 고도 데이터를 조회하여 전방의 오르막을 자동으로 감지합니다.

ClimbPro vs PreClimb

ClimbProPreClimb
경로 필요필수 (GPX)불필요
데이터 소스GPX 파일의 고도 데이터서버 고도 API 실시간 조회
감지 범위전체 경로전방 약 5km
오프라인가능불가 (네트워크 필요)
활성화 조건경로 불러오기경로 없음 + 네트워크 연결

작동 방식

  • 경로를 불러오지 않은 상태에서 주행을 시작하면 자동으로 활성화됩니다
  • 주행 중 일정 거리(200~1,000m)마다 서버에 고도 프로필을 조회합니다
  • 오르막이 감지되면 조회 간격이 짧아지고(200m), 평지에서는 간격이 넓어집니다(1,000m)
  • 전방 최대 3개의 오르막 정보가 표시됩니다
  • 오르막 시작 지점 200m 이내에 도달하면 자동으로 진행 상태가 표시됩니다
계획 없이 자유롭게 달리더라도 PreClimb 덕분에 갑작스러운 오르막에 당황하지 않습니다. 네트워크만 연결되어 있으면 별도 설정 없이 자동으로 작동합니다.
PreClimb은 네트워크가 필요합니다. 데이터 절약 설정에서 네트워크를 차단하면 작동하지 않습니다.
