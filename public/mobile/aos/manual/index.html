처음 사용자를 위한 빠른 시작

O-Rider를 처음 설치하셨나요? 아래 5단계만 따라하면 바로 라이딩을 시작할 수 있습니다.

1
앱 설치 후 권한 허용
첫 실행 시 위치와 블루투스 권한을 허용하세요. 백그라운드 위치도 허용하면 화면이 꺼져도 기록이 이어집니다.
2
센서 연결 (선택사항)
속도/케이던스/심박 센서가 있다면 설정 → 센서에서 페어링합니다. 센서가 없어도 GPS만으로 속도와 거리를 측정할 수 있습니다.
3
데이터 화면 꾸미기 (선택사항)
설정 → 데이터 필드에서 화면에 표시할 데이터를 선택합니다. 기본 구성 그대로 사용해도 좋습니다.
4
주행 시작
메인 화면 하단의 시작 버튼을 탭하면 기록이 시작됩니다. 좌우 스와이프로 데이터/지도 화면을 전환하세요.
5
주행 종료 후 확인
일시정지 버튼을 탭하면 재개/정지 버튼이 나타납니다. 정지 버튼을 탭하면 주행이 종료되고, 거리·시간·속도 등 요약이 표시됩니다. Strava 연동 시 자동 업로드됩니다.
더 자세한 내용은 아래 각 섹션에서 확인하세요. 센서 연결은 4. 센서 연결, 화면 꾸미기는 3.2 레이아웃 에디터를 참고하세요.
