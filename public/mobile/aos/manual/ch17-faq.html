17. 문제 해결 (FAQ)

라이딩 중 문제가 생기면 아래를 먼저 확인해 보세요.

센서가 검색되지 않아요

  • 센서를 깨워주세요 — 크랭크를 돌리거나, 심박 센서를 가슴에 착용하거나, 센서의 버튼을 눌러 절전 모드를 해제합니다
  • 블루투스가 켜져 있는지 확인하세요
  • 다른 앱(Garmin Connect 등)에서 해당 센서가 연결 중이면 먼저 해제하세요. BLE 센서는 동시에 하나의 앱에만 연결됩니다
  • 센서와의 거리가 너무 멀지 않은지 확인하세요 (BLE는 보통 10m 이내)

센서 연결이 자주 끊겨요

  • 센서 배터리를 교체해 보세요 — 가장 흔한 원인입니다
  • 폰 케이스가 금속 재질이면 BLE 신호를 차단할 수 있습니다
  • 여러 BLE 기기가 동시에 연결되어 있으면 간헐적 끊김이 발생할 수 있습니다

GPS 속도/거리가 부정확해요

  • 높은 건물 사이나 나무 밀집 지역에서는 GPS 정확도가 떨어질 수 있습니다
  • 배터리 절약 → GPS 모드가 "고정밀"인지 확인하세요 ("균형" 모드는 정확도가 낮음)
  • 보다 정확한 속도를 원하면 CSC 속도 센서를 연결하세요 — 바퀴 회전 기반이라 GPS보다 정확합니다
  • 앱 시작 직후에는 GPS 위성 수신에 몇 초 걸릴 수 있습니다. GPS 정확도 필드를 확인하여 10m 이내가 될 때까지 기다려 보세요

화면이 꺼지면 기록이 멈춰요

  • Android 설정에서 O-Rider 앱의 백그라운드 위치 권한이 "항상 허용"인지 확인하세요
  • 배터리 최적화에서 O-Rider를 제외하세요 (제조사별 설정이 다릅니다)
  • Samsung: 설정 → 배터리 → 백그라운드 사용 제한 → O-Rider 제외
  • Xiaomi: 설정 → 앱 → O-Rider → 배터리 절약 → 제한 없음
  • 배터리 절약 → 화면 자동꺼짐을 "항상 켜짐"으로 설정하는 것도 방법입니다

Strava 업로드가 안 돼요

  • 설정 → 계정에서 Strava 연결 상태를 확인하세요. "연결 해제" 후 다시 연결해 보세요
  • 데이터 절약 → Strava 업로드가 "오프라인"으로 되어 있지 않은지 확인하세요
  • 네트워크 연결을 확인하세요

ClimbPro가 작동하지 않아요

  • GPX 경로를 먼저 불러와야 합니다. 경로 없이는 오르막 감지가 불가능합니다
  • 경로에 오르막이 없거나 Cat 4 미만(점수 8,000 미만)의 짧은 경사는 감지되지 않습니다
  • 현재 위치가 경로와 너무 멀리 떨어져 있으면 오르막에 진입해도 감지되지 않을 수 있습니다

앱이 비정상 종료돼요

  • 설정 → 고급 → 크래시 로그에서 오류 내역을 확인할 수 있습니다
  • 앱을 최신 버전으로 업데이트해 주세요
  • 문제가 지속되면 크래시 로그를 개발자에게 공유해 주세요
