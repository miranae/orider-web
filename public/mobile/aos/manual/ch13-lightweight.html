13. 경량 모드 (그룹 전용)

장거리 그룹 라이딩에서 배터리가 걱정될 때 사용하는 모드입니다. 화면에 최소한의 정보(지도와 커뮤니케이션 바)만 표시하고, GPS 업데이트 빈도를 줄여 배터리와 데이터를 절약합니다. 동시에 SOS, 핑, 프리셋 메시지로 그룹 멤버와 소통할 수 있습니다.

활성화 방법

  1. 먼저 그룹에 참여합니다 (8.2 그룹 라이딩 참고)
  2. 설정 → 고급 → 경량 모드에서 토글을 켭니다
  3. 주행 화면이 최소 UI로 전환됩니다
그룹에 참여하지 않은 상태에서는 경량 모드를 활성화할 수 없습니다.

GPS 간격 자동 조절

배터리를 절약하기 위해 상황에 따라 GPS 업데이트 간격이 자동 조절됩니다.

상태GPS 간격
화면 꺼짐 + 뷰어 없음10초 (배터리 절약)
화면 켜짐 또는 뷰어 있음2초 (높은 정확도)

경량 모드에서 표시되는 화면

  • 최소 UI — 지도와 커뮤니케이션 바만 표시
  • 커뮤니케이션 바 — SOS, 핑, 프리셋 메시지 전송 가능
그룹을 탈퇴하면 경량 모드가 자동으로 비활성화됩니다.

13.1 그룹 메시징

경량 모드가 활성화된 상태에서 화면 하단의 커뮤니케이션 바를 통해 사전 정의된 메시지를 그룹 멤버에게 전송할 수 있습니다. 라이딩 중 긴 문자를 타이핑할 필요 없이, 한 번의 탭으로 상황을 알릴 수 있습니다.

유형메시지용도
SOS긴급 구조 요청2초 길게 누르기로 전송, 전원에게 경고 표시
SOS 해제구조 요청 취소SOS 전송 후 즉시 취소 가능
핑위치 알림현재 위치를 그룹에 알림
정지멈춤 상태잠시 정차 알림
펑크타이어 펑크펑크 수리 중 알림
카페 대기카페에서 대기휴식 장소 알림
먼저 출발먼저 갈게요선행 출발 알림
쿨다운: SOS 10초, 핑/프리셋 5초. SOS 해제는 쿨다운 없이 즉시 전송 가능합니다.
긴 그룹 라이딩에서는 뒤처진 멤버가 "정지" 또는 "카페 대기"를 보내면 선두 그룹이 속도를 조절할 수 있어 편리합니다.
