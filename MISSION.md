# 미션

Orider는 라이더가 자신의 활동, 훈련 부하, 코스, 회복 신호를 더 잘 이해하고 활용하도록 돕는 제품입니다. 영문 문서는 [MISSION-en.md](MISSION-en.md)를 참고하세요.

## 우리가 만드는 것

- 라이딩과 운동 기록을 보기 좋은 분석 화면
- 파워, 심박, zone, fitness, readiness를 연결한 훈련 인사이트
- 코스, 세그먼트, 그룹 이벤트, 커뮤니티 흐름
- 개인 데이터 API와 recipe를 통한 소유자 중심 자동화

## 기준

1. 사용자의 개인 데이터는 사용자의 것입니다.
2. 기본 공유 상태는 안전해야 합니다.
3. 분석은 근거와 단위를 분명히 해야 합니다.
4. UI는 반복 사용에 편하고 모바일에서도 견고해야 합니다.
5. 공개 저장소는 기여하기 쉬워야 하지만 운영 secret과 private backend는 노출하지 않습니다.

## 오픈소스 방향

이 저장소는 Orider Web 프론트엔드와 재사용 가능한 TypeScript 유틸리티, 문서, recipe를 공개합니다. broad third-party platform은 단계적으로 열고, 먼저 owner-only Personal Data API와 안전한 Creator Showcase를 중심으로 확장합니다.
